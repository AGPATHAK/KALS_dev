After Practice

Run These Commands

After you finish a practice tab and stop the Playwright launcher with Ctrl+C, run these in the terminal to ingest the latest data and refresh the next recommendation.

Ingest Latest Practice Data
.venv/bin/python pipeline/ingest_events.py
Generate and Deliver the Next Recommendation
.venv/bin/python pipeline/deliver_recommendation_handoff.py --save-run
Optional: Inspect The Updated Evaluation
.venv/bin/python pipeline/evaluate_guided_sessions.py --refresh-views

How This Fits

The coach hub reads the current recommendation handoff from browser storage and launches the right app from the browser.

Recommendation generation and data ingest still happen through the Python pipeline for now, but actual practice can already feel much less terminal-driven.
